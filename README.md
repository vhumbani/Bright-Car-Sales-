# Bright-Car-Sales
This repository contains detailed Car sales analysis from 1st January 2023 to 30 September 2023. The goal is to understand revenue by dayname,car make and fuel type.

# Sales/Revenue  were determined by:
* Dayname
* Car make
* Fuel type
* Body type
* Region
* Colour

Summary of revenue by colour
* Black 22.63%
* White 20'67%
* Gray -15.2%
* Silver 12.92%
* Blue 7.69%
* Red 7.27%

# Recomendation on how to increase revenue
* Intoduce discount to cash buyers.
* Increase referal amount.
* Introduce brand new models.
